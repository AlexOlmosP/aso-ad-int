# Conversion Rate Analysis Skill

## CR Formula
Conversion Rate = Total Downloads / Total Page Views

## Elements That Impact Conversion (by importance)

### High Impact (Visual - first 3 seconds)
1. **Icon** — First visual element users see. Bold colors, distinctive design, brand identity
2. **Screenshots (first 3)** — Largest visual area. Must show key features with short CTAs
3. **Video Preview** — Auto-plays in App Store, shows real functionality
4. **Feature Graphic** (GP only) — Shown above screenshots when video present

### Medium Impact (Trust Signals)
5. **Average Rating** — Above 4.1 stars recommended (PICKASO benchmark)
6. **Rating Volume** — Higher volume = more trust = higher conversion
7. **Install Count** (GP) — Social proof indicator

### Lower Impact (Text - less than 2% read full description)
8. **App Name/Title** — First text element, must communicate value
9. **Subtitle / Short Description** — Secondary text, reinforces value proposition
10. **Description (first lines)** — Only visible portion matters for conversion

## Priority Classification
- **High Priority**: Element is missing, broken, or significantly below best practice
- **Medium Priority**: Element exists but could be optimized
- **Low Priority**: Element is present and adequate, minor improvements possible

## Recommendation Generation Rules
- Be specific and actionable (not "improve your icon" but "test a darker background color to stand out in search results")
- Reference the current state of the element
- Suggest A/B testing for visual changes
- Consider the competitive landscape in the category
- Note that 40% of users spend less than 3 seconds on the listing
